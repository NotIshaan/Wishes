# 🎉 Celebration Web App

A fun and personalized celebration website built with **React** and **Tailwind CSS**, featuring surprise animations, heartfelt messages, and interactive visuals for birthdays and anniversaries.

---

## ✨ Features

- 🎁 "Tap for Surprise" interaction with:
  - Confetti animation
  - Animated letter reveal
  - Pop-up photo transitions with bouncy effect
- 🎂 Birthday Pages
  - Beautiful SVG icons
  - Personalized messages
  - Popcorn-style photo reveal with tilt effect
- 💍 Anniversary Page
  - Photo carousel using `react-slick`
  - Short messages under each photo

---

## 🧰 Tech Stack

- **React** (Vite or CRA)
- **Tailwind CSS**
- **React Confetti**
- **React Slick** (carousel)
- **Local Images & SVG Icons**
